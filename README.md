# solana-organic-volume-bot
Organic Looking Solana Volume Bot in raydium, pumpfun, meteora, bonkfun, pump swap.

## Features:
- Anti-MEV execution: 3 buy + 2 sell in one bundle
- Not same volume wallets at all, every tx is executing in different wallets
- Organic Looking
